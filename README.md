## Commit Convention

**Chore**: build м—…л¬ҙ, package manager мҲҳм •, package кҙҖлҰ¬мһҗ кө¬м„ұ update л“ұ<br/>
**Feat**: мғҲлЎңмҡҙ кё°лҠҘ м¶”к°Җ<br/>
**Fix**: bug мҲҳм •<br/>
**Refactor**: code refactoring<br/>
**Comment**: н•„мҡ”н•ң мЈјм„қ м¶”к°Җ л°Ҹ ліҖкІҪ<br/>
**Style**: code fomat ліҖкІҪ, semi colon лҲ„лқҪ<br/>
**Design**: UI Design ліҖкІҪ (styled-component)<br/>
**Rename**: folder, file мқҙлҰ„ мҲҳм • л°Ҹ мқҙлҸҷ<br/>
**Remove**: file мӮӯм ң<br/>
**Docs**: л¬ём„ң мҲҳм •<br/>
**Test**: test code<br/>
